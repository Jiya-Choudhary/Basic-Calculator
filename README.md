# 🧮 Basic Calculator

A simple command-line calculator built using Python that performs basic arithmetic operations such as addition, subtraction, multiplication, and division with continuous user interaction and error handling.

## Features

* Addition (+)
* Subtraction (-)
* Multiplication (*)
* Division (/)
* Division-by-zero handling
* Continuous calculations until user exits
* Simple and interactive command-line interface

## Technologies Used

* Python 3

## How to Run

```bash
python calculator.py
```
